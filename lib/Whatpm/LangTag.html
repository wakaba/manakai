NAME

Whatpm::LangTag - Language Tag Parsing, Conformance Checking, and Normalization

SYNOPSIS

  use Whatpm::LangTag;
  $tag = Whatpm::LangTag->normalize_rfc5646_langtag ($tag);
  $parsed = Whatpm::LangTag->parse_rfc4646_langtag ($tag, $onerror);
  Whatpm::LangTag->check_rfc4646_langtag ($parsed, $onerror);
  Whatpm::LangTag->check_rfc3066_language_tag ($tag, $onerror);

DESCRIPTION

The the Whatpm::LangTag manpage module contains methods to handle language tags as defined by BCP 47. It can be used to parse, validate, or normalize language tags according to relevant standard.

METHODS

$tag = Whatpm::LangTag->normalize_rfc5646_langtag ($tag_orig)

Normalize a language tag by folding cases, following RFC 5646 2.1. Note that this method does not replace any subtag into its preferred alternative.

$parsed = Whatpm::LangTag->parse_rfc4646_langtag ($tag, $onerror)

Parses a language tag into subtags. This method interprets a language tag using the definition in RFC 4646. Note that RFC 4646 is obsoleted by RFC 5646.

Any parse errors and warnings are reported to the second argument, i.e. the $onerror code reference. Errors are reported in the format similar to the Whatpm::HTML manpage's $onerror callbacks.

The method returns a hash reference, which contains one or more key-value pairs from the following list:

language

The language subtag.

extlang

The extlang subtags.

script

The script subtag.

region

The region subtag.

variant

The variant subtags.

extension

The extension subtags.

privateuse

The privateuse subtags.

illegal

Illegal (syntactically non-conforming) string fragments.

grandfathered

"Grandfathered" language tag.

Whatpm::LangTag->check_rfc4646_langtag ($parsed, $onerror)

Finds conformance errors from a parsed langauge tag with regard to RFC 4646. Note that RFC 4646 is obsoleted by RFC 5646.

Any errors and warnings are reported to the second argument, i.e. the $onerror code reference. Errors are reported in the format similar to the Whatpm::HTML manpage's $onerror callbacks.

This method does not report any parse erros, as this method receives a parsed langauge tag.

Whatpm::LangTag->check_rfc3066_language_tag ($tag, $onerror)

Parses and finds conformance errors in a language tag (a string) with regard to RFC 3066. Note that RFC 3066 is obsoleted by RFC 4646.

Any errors and warnings are reported to the second argument, i.e. the $onerror code reference. Errors are reported in the format similar to the Whatpm::HTML manpage's $onerror callbacks.
