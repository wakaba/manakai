Whatpm::CSS::Tokenizer

CSS Tokenizer

DESCRIPTION

The Whatpm::CSS::Tokenizer module provides a CSS syntax tokenizer. It can tokenize a string written in a language using the CSS syntax, such as CSS, Selectors, and Media Queries.

SEE ALSO

CSS 2.1 <http://www.w3.org/TR/CSS2/>.

Selectors <http://www.w3.org/TR/selectors>.

Whatpm::CSS::Parser, Whatpm::CSS::SelectorsParser, Whatpm::CSS::MediaQueryParser.
