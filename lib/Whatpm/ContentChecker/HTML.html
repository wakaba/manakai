NAME

Whatpm::ContentChecker::HTML - DOM Conformance Checker - HTML

DESCRIPTION

The Whatpm::ContentChecker::HTML implements conformance checking of HTML vocabulary, i.e. elements and attributes in the HTML namespace as well as attributes in no namespace of HTML elements.

For more information, see the Whatpm::ContentChecker manpage.

SEE ALSO

Web Applications 1.0 <http://www.whatwg.org/specs/web-apps/current-work/complete.html> - the HTML standard, including HTML5.

manakai's Conformance Checking Guideline for Obsolete HTML Elements and Attributes <http://suika.fam.cx/www/markup/html/exts/manakai-obsvocab> - Requirements for non-standard elements and attributes.

the Whatpm::ContentChecker manpage.
