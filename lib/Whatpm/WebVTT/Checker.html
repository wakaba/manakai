NAME

Whatpm::WebVTT::Checker - A WebVTT conformance checker

SYNOPSIS

  use Whatpm::WebVTT::Checker;
  
  $checker = Whatpm::WebVTT::Checker->new;
  $checker->onerror ($error_handler);
  $checker->check_track ($track);

DESCRIPTION

The Whatpm::WebVTT::Checker module, in conjunction with the Whatpm::WebVTT::Parser manpage, forms a conformance checker for WebVTT.

METHODS

Following methods can be used to check the conformance of WebVTT documents:

$parser = Whatpm::WebVTT::Checker->new

Return a new instance of the WebVTT checker.

$parser->onerror ($new_code)
$code = $checker->onerror

Get or set the error handler. See the Whatpm::Errors manpage for more information.

$checker->check_track ($track)

Check a TextTrack (the Message::DOM::TextTrack manpage) object representing a WebVTT file. Any conformance error is reported via the error handler specified to the onerror attribute.

In addition, following methodis provided as lower-level APIs of the parser: check_text_document_fragment.
