Whatpm::HTML

Old HTML parser API

DESCRIPTION

The Whatpm::HTML module provides an out-of-date API to HTML parser for backward compatibility. Use Whatpm::HTML::Parser and Whatpm::HTML::Serializer for new applications.

SEE ALSO

Whatpm::HTML::Parser, Whatpm::HTML::Serializer.
