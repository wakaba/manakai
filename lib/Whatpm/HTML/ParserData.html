NAME

Whatpm::HTML::ParserData - Data for HTML parser

DESCRIPTION

The Whatpm::HTML::ParserData module contains data for HTML parser, extracted from the HTML Living Standard.

CONSTANTS

Following constants for namespace URLs are defined (but not exported): HTML_NS (HTML namespace), SVG_NS (SVG namespace), MML_NS (MathML namespace), XML_NS (XML namespace), XMLNS_NS (XML Namespace namespace), and XLINK_NS (XLink namespace).

VARIABLES

Following data from the HTML specification are included:

$MathMLTextIntegrationPoints
$MathMLHTMLIntegrationPoints
$SVGHTMLIntegrationPoints
$ForeignContentBreakers
$MathMLAttrNameFixup
$SVGAttrNameFixup
$ForeignAttrNamespaceFixup
$SVGElementNameFixup
$NamedCharRefs
$CharRefReplacements
$NoncharacterCodePoints
$ObsoletePermittedDoctypes
$QuirkyPublicIDPrefixes
$QuirkyPublicIDs

SEE ALSO

HTML Living Standard <http://www.whatwg.org/specs/web-apps/current-work/>.
