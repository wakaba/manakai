=head1 NAME

Whatpm::HTML::InputStream - Input stream integration for HTML/XML parsers

=head1 DESCRIPTION

The C<Whatpm::HTML::InputStream> module implements handling of input
stream for HTML/XML parsers (tokenizers), including the coordination
with character encoding layer.  This module is not intended to be used
standalone.

=head1 SEE ALSO

L<Whatpm::HTML::Parser>, L<Whatpm::XML::Parser>.

L<Whatpm::HTML::Tokenizer>.

=head1 SPECIFICATIONS

=over 4

=item [HTML]

HTML Living Standard - The input byte stream
<http://www.whatwg.org/specs/web-apps/current-work/#the-input-byte-stream>.

=item [ENCODING]

Encoding Standard
<http://dvcs.w3.org/hg/encoding/raw-file/tip/Overview.html>.

=back
