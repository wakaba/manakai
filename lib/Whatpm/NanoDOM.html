NAME

Whatpm::NanoDOM - A Non-Conforming Implementation of DOM Subset

DESCRIPTION

The Whatpm::NanoDOM module contains a non-conforming implementation of a subset of DOM. It is the intention that this module is used only for the purpose of testing the Whatpm::HTML module.

See source code if you would like to know what it does.
