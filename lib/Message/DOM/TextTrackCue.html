NAME

Message::DOM::TextTrackCue - DOM |TextTrackCue| object

DESCRIPTION

The Message::DOM::TextTrackCue class implements the DOM TextTrackCue interface.

METHODS

The Message::DOM::TextTrackCue object has following methods:

$track = $cue->track

Return the associated TextTrack object, if any, or undef. [HTML]

$id = $cue->id
$cue->id ($new_id)

Get or set the identifier of the cue. [HTML]

$seconds = $cue->start_time
$cue->start_time ($new_seconds)

Get or set the start time of the cue. [HTML]

$seconds = $cue->end_time
$cue->end_time ($seconds)

Get or set the end time of the cue. [HTML]

$bool = $cue->pause_on_exit
$cue->pause_on_exit ($new_bool)

Get or set the pause-on-exit flag of the cue. [HTML]

$value = $cue->vertical
$cue->vertical ($new_value)

Get or set the writing direction of the cue. [HTML]

$bool = $cue->snap_to_lines
$cue->snap_to_lines ($new_bool)

Get or set the snap-to-lines flag of the cue. [HTML]

$int = $cue->line
$cue->line ($new_int)

Get the computed line position, or set the line position, of the cue. [HTML]

$int = $cue->size
$cue->size ($new_int)

Get or set the text position of the cue. [HTML]

$value = $cue->align
$cue->align ($new_value)

Get or set the alignment of the cue. [HTML]

$text = $cue->text
$cue->text ($new_text)

Get or set the text of the cue, in raw unparsed format. [HTML]
