Message::DOM::Node

DOM Node interface

DESCRIPTION

The Message::DOM::Node implements the DOM Node interface.

METHODS

The Node object has following methods:

...
$url = $node->manakai_get_child_namespace_uri ($tag_name)

Return the default namespace URL for the tag name, using the node as the context. [MANAKAI]
