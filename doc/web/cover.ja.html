manakai ―― インターネット・メッセージと XML のための Perl ライブラリ

manakai は、 RFC 2822 系インターネット・メッセージと XML を処理するための Perl 用オブジェクト指向モジュール群です。

manakai は、 DOM 界面を使って XML 文書を操作することができます。 manakai の DOM 対応状況は次の表の通りです。

モジュール 版 補足説明
機能 水準 3 (manakai 独自)

次のすべてを実装:

  • ImplementationRegistry
  • ImplementationSource
  • ImplementationList
  • MinimumImplementation
  • GetFeature
共通 LS 水準 3 (manakai 独自)

次のすべてを実装:

  • GLSImplementation
中核 水準 1, 2, 3
  • whole_text, replace_whole_text, normalize_document, rename_node, import_node は未実装
  • normalize で設定引数 normalize-character は未実装
  • 一部の操作は specified を自動的に真にしない
XML 水準 1, 2, 3
  • internal_subset は未実装
XDoctype 定義 水準 3 案 (manakai 独自)

次の機能を実装:

  • DocumentTypeDefinition
  • ElementTypeDefinition
  • AttributeDefinition
XDoctype 宣言 水準 3 案 (manakai 独自)

次の機能のみ実装:

  • DocumentTypeDeclaration
LS 水準 3

次の機能のみ実装:

  • LSInput
  • LSOutput
  • LSParser (同期読込みのみ、ほとんどの設定引数は未実装)
単純 LS 第 1.0 版案 (manakai 独自)

次のすべてを実装:

  • SimpleXMLSerializer
URI 共通 第 4.0 版 (manakai 独自)

次のすべてを実装:

  • URIImplementation
  • URIReference

manakai の RFC 2822 系インターネット・メッセージを操作するためのモジュール群は、 現在整理中です。
