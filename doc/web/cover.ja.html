インターネット・メッセージと XML のための Perl モジュール群

紹介

manakai は、 RFC 2822 系インターネット・メッセージと XML を処理するための Perl 用オブジェクト指向モジュール群です。

この文書は準備中です。 SuikaWiki:manakai もあわせてご覧下さい。

manakai は、 DOM 界面を使って XML 文書を操作することができます。 manakai の DOM 対応状況は次の各表をご覧下さい。

  • manakai が実装する DOM モジュール
  • manakai 0.4 の変更点
  • manakai 0.3.* の変更点

他のモジュールへの依存性

Perl 5.8.7
古い版の Perl では動作しません。 5.8.4 くらいでも動くかもしれませんが、 未確認です。
CharClass
XML 構文解析器を使用する場合に Char::Class::XML が必要です。

インストール

  1. 配布ファイルを入手します。
  2. 配布ファイルを適切な場所で展開します。 lib/ 以下の Perl モジュール・ファイルを Perl が見つけられる場所 (@LIB で指定されたディレクトリ) に置くだけで構いません。
