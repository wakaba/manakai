manakai

インターネット・メッセージと XML のための Perl モジュール群

紹介

manakai は、 RFC 2822 系インターネット・メッセージと XML を処理するための Perl 用オブジェクト指向モジュール群です。

この文書は準備中です。 SuikaWiki:manakai もあわせてご覧下さい。

manakai は、 DOM 界面を使って XML 文書を操作することができます。 manakai の DOM 対応状況は次の各表をご覧下さい。

  • manakai が実装する DOM モジュール

manakai の RFC 2822 系インターネット・メッセージを操作するためのモジュール群は、 現在整理中です。
