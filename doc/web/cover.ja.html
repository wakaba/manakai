インターネット・メッセージと XML のための Perl モジュール群

これは何?

manakai は、 RFC 2822 系インターネット・メッセージと HTML 文書、XML 文書、 CSS スタイル・シートを処理するための Perl 用オブジェクト指向モジュール群です。

manakai は、標準規格に沿った DOM 界面を使って HTML 文書、 XML 文書、CSS スタイル・シートを操作することができます。manakai の DOM 対応状況は次の表をご覧下さい。

  • manakai が実装する DOM モジュール

manakai の RFC 2822 系インターネット・メッセージを操作するためのモジュール群は、 現在整理中です。

manakai-core を用いた応用

  • Harusame, multilingual Web page management tool.
  • Bunshin.pm

    Web 上の連続する資源 (掲示板の記事のようなもの) を切り出して RFC 822 形式のメッセージにするエンジン部分。

  • すいかわり

    Bunshin.pm を使って、電子ニュースに記事を投稿する script。 (cron とかから呼び出して使う。)

  • WebHACC (Web hypertext application conformance checker) (See also about WebHACC).
  • mkcommitfeed.pl

他のモジュールへの依存性

Perl 5.8.7 or later
古い版の Perl では動作しません。 5.8.4 くらいでも動くかもしれませんが、 未確認です。
CharClass
DOM 実装を使用する場合に Char::Class::XML が必要です。

インストール

  1. 配布ファイルを入手します。
  2. 配布ファイルを適切な場所で展開します。 lib/ 以下の Perl モジュール・ファイルを Perl が見つけられる場所 (@LIB で指定されたディレクトリ) に置くだけで構いません。
