manakai 0.3.3 の機能

この文書は準備中です。 SuikaWiki:manakai もあわせてご覧下さい。

manakai 0.3.3 は、次の表に示す DOM モジュールを実装しています。

モジュール 版 補足説明
機能 水準 3 (manakai 独自)

次のすべてを実装:

  • ImplementationRegistry
  • ImplementationSource
  • ImplementationList
  • MinimumImplementation
  • GetFeature
共通 LS 水準 3 (manakai 独自)

次のすべてを実装:

  • GLSImplementation
中核 水準 1, 2, 3

次を除き実装済:

  • whole_text, replace_whole_text, normalize_document, rename_node, import_node は未実装
  • normalize で設定引数 normalize-character は未実装
  • 一部の操作は specified を自動的に真にしない
XML 水準 1, 2, 3

次を除き実装済:

  • internal_subset は未実装
XDoctype 定義 水準 3 案 (manakai 独自)

次のすべてを実装:

  • DocumentTypeDefinition
  • ElementTypeDefinition
  • AttributeDefinition
XDoctype 宣言 水準 3 案 (manakai 独自)

次に挙げるもののみ実装:

  • DocumentTypeDeclaration
XDP 第 1.0 版案 (manakai 独自)
LS 水準 3

次に挙げるもののみ実装:

  • LSInput
  • LSOutput
  • LSParser (同期読込みのみ、ほとんどの設定引数は未実装)
単純 LS 第 1.0 版案 (manakai 独自)

次のすべてを実装:

  • SimpleXMLSerializer
URI 共通 第 4.0 版 (manakai 独自)

次のすべてを実装:

  • URIImplementation
  • URIReference
探索 水準 2 NodeIterator は未実装。
(manakai 独自) SerialWalker。
Atom 第 1.0 版案 (manakai 独自)
SuikaWiki/0.9 構文解析 第 1.0 版案 (manakai 独自)
H2H 構文解析 第 1.0 版案 (manakai 独自)
