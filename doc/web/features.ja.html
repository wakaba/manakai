manakai 0.4.0 の機能

この文書は準備中です。 SuikaWiki:manakai もあわせてご覧下さい。

manakai 0.4.0 は、次の表に示す DOM モジュールを実装しています。

モジュール 版 補足説明
中核 水準 1, 2, 3

ほぼ実装済

  • replace_whole_text, normalize_document, rename_node, import_node は未実装
  • normalize で設定引数 normalize-character は未実装
  • 一部の操作は specified を自動的に真にしない
  • 設定引数のうち error-handler と schema-type 以外 は未実装
  • NameList は未実装

replace_whole_text と import_node と specified はいつか実装する予定です。 それ以外の実装の予定はありません。
